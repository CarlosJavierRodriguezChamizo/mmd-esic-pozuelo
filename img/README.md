# Carpeta de imágenes

Contiene los recursos gráficos del sitio. Todos son **SVG** (vectoriales, ligeros y
autocontenidos, sin peticiones externas):

| Archivo | Uso |
|---|---|
| `logo.svg` | Logotipo principal (cabecera, versión clara) |
| `logo-white.svg` | Logotipo para fondos oscuros (footer) |
| `favicon.svg` | Icono de la pestaña del navegador |
| `hero-bg.svg` | Fondo degradado de la sección Hero |
| `hero-app.svg` | Mockup de la app en el Hero |
| `icon-goals.svg` | Icono USP — Objetivos |
| `icon-auto.svg` | Icono USP — Ahorro automático |
| `icon-learn.svg` | Icono USP — Microlearnings |
| `icon-secure.svg` | Icono USP — Seguridad |
| `badge-appstore.svg` | Botón de descarga App Store |
| `badge-googleplay.svg` | Botón de descarga Google Play |

Para sustituir cualquier gráfico por una imagen real (`.jpg`, `.png`, `.webp`),
guárdala aquí y actualiza la ruta correspondiente en el HTML/CSS.
